## 2. Critical Incomplete (Blocking Features)

These items block core functionality expected by the user.

### 1. OpenSim Simulation Logic Missing
*   **Location:** `engines/physics_engines/opensim/python/opensim_gui.py`
*   **Method:** `OpenSimGolfGUI.run_simulation` calls `self.model.run_simulation()`
*   **Issue:** The method catches `NotImplementedError` and displays "OpenSim simulation is not yet fully implemented". This indicates the backend `opensim_golf.core` or the GUI integration is unfinished.
*   **Impact:** **High (5/5)**. Users cannot run OpenSim simulations via the GUI, rendering the "OpenSim Golf Interface" largely non-functional for simulation tasks.
*   **Action:** Created Issue `ISSUE_OPEN_SIM_INCOMPLETE`.

### 2. MuJoCo Live Analysis Configuration Disconnect
*   **Location:** `engines/physics_engines/mujoco/python/humanoid_launcher.py`
*   **Method:** `RemoteRecorder.set_analysis_config`
*   **Issue:** The method is a `pass` with comment "Config driven by simulation loop". However, `LivePlotWidget` calls this method to update analysis settings from the UI. Since `RemoteRecorder` does not forward this to the remote simulation process, UI controls for live analysis have no effect.
*   **Impact:** **Medium-High (4/5)**. Features implying live control of analysis metrics are broken/misleading.
*   **Action:** Created Issue `ISSUE_MUJOCO_CONFIG_DISCONNECT`.
