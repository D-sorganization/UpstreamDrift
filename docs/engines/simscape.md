# Simscape Multibody Models

## Overview
These models utilize MATLAB's Simscape Multibody technology to provide a block-diagram environment for 3D mechanical systems.

## Key Features in Suite
- **2D Golf Model**: Simplified planar analysis.
- **3D Golf Model**: Full spatial biomechanical model.
- Integration with MATLAB's control and analysis toolboxes.

## Usage
Located in `engines/Simscape_Multibody_Models/`.
Requires MATLAB. Run `setup_golf_suite()` in MATLAB to initialize paths.
